# Welcome

This blog is generated from Markdown files inside the `posts/` folder.

## How publishing works

- Add a new `.md` file in `posts/`
- Commit and push it to GitHub
- Cloudflare Pages rebuilds and publishes the updated site

You can also use **bold text**, *italics*, inline `code`, and fenced code blocks.

```js
console.log("New post published");
```
